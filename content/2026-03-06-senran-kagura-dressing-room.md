## はじめに：「更衣室」という名の小宇宙

2026年3月6日。夜の静寂の中で、私は再び CoreBook X の電源を入れます。
今日、私が向き合っているのは、爆乳ハイパーバトル『閃乱カグラ Burst Re:Newal』。多くの人がこのゲームに抱くイメージは、華麗なアクションや「命懸けで脱ぎに来る」刺激的な演出でしょう。しかし、真の忍（プレイヤー）が最も長い時間を費やす場所は、戦場ではありません。

それは、「更衣室」です。

単なるキャラクターの衣装を着せ替えるだけの場所……そう侮るなかれ。そこには、色彩、ライティング、ポーズ、そして空気感までもをコントロールできる、無限の表現の可能性が秘められています。今日は、CoreBook X の高精細ディスプレイをキャンバスに、私が辿り着いた「究極のスクリーンショット術」とその美学について、技術的な側面を交えながら語り尽くしたいと思います。

## 1. 2Kディスプレイが解き放つ、「質感」の深淵

まず語るべきは、CoreBook X が搭載する 14インチ 2160×1440 解像度のディスプレイが、いかにこのゲームの「更衣室」体験を変えたか、ということです。

### 髪の毛一本、布の繊維一本へのこだわり
『閃乱カグラ Burst Re:Newal』のキャラクターモデルは、シリーズの中でも非常に高いクオリティを誇ります。
一般的な 1080p（フルHD）の環境では潰れてしまいがちな、髪の毛のハイライトの繊細なグラデーションや、忍装束の布地の質感、そして肌に落ちる影の柔らかさ。これらが、CoreBook X の 2K ディスプレイ上では驚くほど鮮明に、かつ滑らかに描き出されます。

特に、3:2 という画面比率は、キャラクターを縦位置で捉える際に絶大な威力を発揮します。16:9 のワイド画面では左右に余白ができがちですが、3:2 であればキャラクターを画面いっぱいに、全身のディテールを損なうことなく収めることができるのです。この「収まりの良さ」こそが、構図を決定する上での第一の快感となります。

## 2. 構図の美学：黄金比と「静止した躍動感」

スクリーンショットは、単なる記録ではありません。それは、ゲーム内の時間を一瞬だけ止める「写真術」です。

### ポージングの選択：内面を写し出す
更衣室では、各キャラクターに様々なポーズを取らせることができます。
焔であれば、その力強さと不敵な笑みを強調する低い構え。飛鳥であれば、若さと正義感を感じさせる伸びやかな立ち姿。
私はまず、キャラクターの性格やその時の衣装のコンセプト（例えば「月夜に舞う暗殺者」や「放課後の休息」など）に合わせて、基本となるポーズを数時間かけて吟味します。

### カメラアングル：FOV（視野角）の魔術
ここで重要なのが、カメラの距離と FOV の調整です。
あえて広角（FOVを広く）にしてキャラクターを煽るように撮れば、躍動感とスケール感が生まれます。逆に、望遠（FOVを狭く）にして少し離れた位置から狙えば、歪みが少なく、ポートレート写真のような端正な美しさが際立ちます。
CoreBook X のトラックパッドは、こうした微細なカメラ操作にもリニアに反応してくれるため、理想のアングルをコンマ数ミリ単位で追い込むことが可能です。

## 3. ライティングと色彩：感情を乗せる光の演出

『Burst Re:Newal』の更衣室には、照明の位置や色温度を微調整できる機能があります（※一部の外部ツールや撮影モードを含む）。

### 三点照明の原則
私はリアルな人物撮影のテクニックを応用し、「キーライト」「フィルライト」「バックライト」のバランスを考えます。
- **キーライト:** 斜め前方から当てて、顔の陰影を際立たせる。
- **バックライト:** 背後から当てることで、髪の輪郭に光のラインを作り、背景からキャラクターを浮き上がらせる。

CoreBook X の IPS パネルは色再現性が非常に高く、sRGB カバー率も優秀なため、こうした繊細な光の色調整がそのまま完成品に反映されます。
「もう少し、冷たい月明かりのような青を足したい」
「夕暮れ時の、寂しげだが温かいオレンジを乗せたい」
指先一つで世界の色が変わるその瞬間、私はクリエイターとしての至福を感じます。

## 4. テクニカル・アプローチ：OBSを活用した最高画質キャプチャ

単に「PrintScreen」を押すだけでは、真の最高画質は得られません。私は CoreBook X の Ryzen 5 7430U のパワーを活かし、OBS Studio をバックグラウンドで走らせながら撮影を行っています。

### OBSの設定：ビットレートの限界へ
Ryzen 5 7430U はマルチスレッド性能に優れているため、ゲームを動かしながら高画質なエンコードを行う余裕があります。
- **エンコーダ:** AMD HW H.264 (AVC)
- **レート制御:** CQP (値: 15〜18)
- **解像度:** 2160x1440（ネイティブ）

あえて動画として数秒間キャプチャし、その中から「瞳のまばたき」や「髪の毛の揺れ」が最も美しい一瞬を切り出す。あるいは、静止画キャプチャとして保存する。
この際、OBS のフィルタ機能を使って、微細なシャープネスや色調補正（LUT）をリアルタイムで掛けることもあります。CoreBook X の性能があれば、こうした「撮影しながらのポストプロセス」も、フレームレートを落とすことなく実行可能です。

## 5. 忍のカスタマイズ：精神性を纏う装束

例えば、傷だらけの忍装束に、あえて不釣り合いな「リボン」を添えてみる。
それは、過酷な戦いの中に身を置きながらも、少女としての心を捨てきれない葛藤の現れ……。
このような「裏設定」を自分の中で構築し、それを視覚化するために小物を配置していく作業は、もはやキャラメイクの域を超え、一種の「文学」に近い行為と言えるでしょう。

CoreBook X で過去のスクリーンショットを見返しながら、新しい衣装の組み合わせをメモする時間は、私にとって至高のヒーリングタイムとなっています。
