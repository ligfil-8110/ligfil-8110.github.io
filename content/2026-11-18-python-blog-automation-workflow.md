## 2. 実践：GitHub ActionsとPythonの「握手」

私は、以下の機能を持つ `deploy_blog.py` を作成した。

*   **自動整形:** Markdownの構文エラー、未リンクの画像を検知。
*   **メタデータ補完:** 日付やSlugをタイトルから自動生成。
*   **一括デプロイ:** コマンド一つで `git add / commit / push` を完遂。
*   **通知連携:** ビルド成功をDiscordへ飛ばし、自作PCの右端に表示。

このレスポンスの良さ。
