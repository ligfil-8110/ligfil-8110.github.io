Summary: せっかく技術的なメモを書くなら見た目にもこだわりたい！ということで、ブログのコード表示を「テックブログ風」に改造してみました。

最近、CoreBook Xのセットアップなんかを記事にしていますが、ソースコードを載せたときに見栄えが「メモ帳」みたいで、ちょっと寂しいなと思っていました。

せっかくの技術メモなので、読みやすく、そして何より「それっぽく」したい！
ということで、今回はコードブロックの見た目をテックブログ風に改造したので、その内容を記録しておきます。

## 今回やったこと

大きく分けて、以下の3つを導入しました。

1. **シンタックスハイライト**（コードに色を付ける）
2. **ダークモード風の装飾**（Monokai風の背景）
3. **コピーボタン**（ワンクリックでコピペできる！）

### 1. コードに色がつくだけでテンションが上がる

今まではただのテキストだったコードですが、`Pygments`というライブラリを導入して、言語に合わせて色が付くようにしました。

```bash
# こんな感じで、コマンドにも色がつくようになりました！
wsl --install
```

やっぱり色がつくと「技術ブログを書いてるぞ！」という感じがして、モチベーションが上がりますね（笑）。

### 2. コピペを快適にする「Copyボタン」

テックブログでよく見る、右上の「Copy」ボタンも自作してみました。
JavaScriptを使って、クリックするとクリップボードにコピーされ、表示が「Copied!」に変わる仕組みです。

これで、自分で自分の記事からコマンドをコピペする時も楽ちんになります。

### 3. CSSで見た目を整える

背景をダークグレーにして、フォントをプログラミングっぽいものに変え、角を丸くして……と、CSSで細かく調整しました。Foundationの標準スタイルと喧嘩してボタンが巨大化するトラブルもありましたが、なんとか力技（!important）で抑え込みました。

## まとめ

今回のカスタマイズで、このブログもだいぶ「テックブログ」らしい風格が出てきた気がします。

中身（記事）も大事ですが、やっぱり自分好みのデザインで書ける環境っていうのは大事ですね。これからも、 CoreBook X を使い倒しながら、気になった技術ネタをどんどんストックしていこうと思います！
