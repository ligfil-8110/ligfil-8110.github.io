## 2. 実践：物理演算による「嘘」の炙り出し

システムの核心部は、以下の二つのモジュールの比較（コンパリゾン）にある。

1.  **申告取得モジュール:** Windows Management Instrumentation (WMI) を叩き、OSが「認識している」CPU名称を取得する。
2.  **実測演算モジュール:** 複雑な浮動小数点演算、および行列積を全コアで10秒間実行。その「実効スループット」を、独自のアルゴリズムで標準的なCPUデータベースと照合する。
