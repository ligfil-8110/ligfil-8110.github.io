真実の咆哮：ベンチマークが証明する「本物の性能」と3月の偽装への決着

序章：数字は嘘をつかない、人間だけが嘘をつく

自作PCの世界において、ベンチマークソフトは「聖書」のような存在だ。 そこには、忖度も、偽装も、言い訳も通用しない。 実行ボタンを押した瞬間に始まる、シリコンへの限界負荷。 数分後、画面に叩き出される一つの数字。 その数字こそが、そのマシンの価値であり、存在理由だ。

3月末、私は絶望の中にいた。 CHUWI CoreBook Xが叩き出したベンチマークスコアは、スペック表から期待される数字の半分にも満たなかった。 BIOSを書き換え、OSを騙し、ユーザーの目を欺こうとしたメーカーの悪意。 私はその「嘘の数字」に、心底嫌気がさしていた。

今日、私はあえて同じベンチマークソフトを起動する。 それは、過去の自分への弔いであり、そして新しい相棒への「洗礼」だ。 嘘のないパーツたちが、嘘のない環境で、どれほどの咆哮を上げるのか。 刮目してほしい。

1. Cinebench 2024：Ryzen 7 7800X3Dの「地力」

まずは、CPUのレンダリング性能を測定するCinebench 2024だ。 3D V-Cacheを搭載した7800X3Dは、本来は「ゲーミング特化」であり、多コアを売りにするレンダリングモンスターではない。しかし、その「地力」は圧倒的だった。

  • Multi Core Score: 1,120 pts
  • Single Core Score: 118 pts

この数字を見た時、私は思わず笑みが溢れた。 CoreBook X（5500U偽装モデル）のマルチコアスコアは、400 pts台を彷徨っていた。 単純計算で、3倍近い性能差だ。 しかも、特筆すべきは「安定感」だ。 ベンチマーク実行中、DeepCool AK620のファンは力強く回り、温度は80℃付近でピタリと止まっている。 スロットリング（熱による性能低下）などという無様な姿は、ここにはない。 これが、正しい冷却と、正しい電力設計に支えられた「誠実なパワー」の証だ。

シングルコアの伸びも素晴らしい。 一つひとつの処理が、淀みなく、瞬時に完了していく。 「8コア16スレッド」という、スペック表に書かれた物理的な事実が、そのまま画面上のバーの伸びとして可視化される。 偽装の影は、もうどこにもない。

2. 3DMark Time Spy：RTX 4070 Ti Super、4Kへの門

続いて、ゲーミング性能の指標である3DMark Time Spyを実行する。 DirectX 12環境下での、GPUの真の実力が問われるテストだ。

  • Graphics Score: 24,850
  • Total Score: 22,120

驚異的な数字だ。 かつての偽装ノートPCが、内蔵グラフィックスで「何とか動いています」という顔をして数千点のスコアを出していたのが、遠い昔の出来事のように思える。 Time Spyのデモシーンが、Hisense 43 U7Nの広大な画面で、144Hz近いフレームレートで滑らかに動き続ける。 カクつきも、ティアリングもない。 ただ、圧倒的なグラフィックの濁流が、私の視界を飲み込んでいく。

この2万4千点というスコアは、4K解像度においても最新のゲームを「最高設定」で遊べるという、確かな保証書だ。 NVIDIAのAD103チップが、16GBのVRAMをフルに活用し、複雑なシェーダーを軽々と処理していく。 その「余裕」こそが、ユーザーに対する最大の誠実さだと私は思う。

3. 3DMark Port Royal：レイトレーシングという「真実の光」

そして、現代のグラフィック技術の極致、レイトレーシング性能を測るPort Royalだ。 光の反射、屈折、影の計算。 ごまかしの効かない物理シミュレーションの世界。

  • Graphics Score: 16,200

CoreBook Xでは、そもそも実行することすら困難だったこのテスト。 RTX 4070 Ti Superは、鏡のような床に映り込む光の筋を、リアルタイムに、そして精緻に描き出した。 レイトレーシング・コア（RTコア）という、物理的な演算ユニットが、その設計通りに火花を散らしている。

私は、この光の煌めきの中に、ようやく「真実」を見つけた気がした。 嘘で固めたスペックでは、この光の一粒さえも正しく描くことはできない。 本物のハードウェアだけが、本物の光を放つことができるのだ。

4. 偽装PCとの決定的対比：なぜ私は「自作」を選んだのか

改めて、3月末のCoreBook Xと、今日の自作PCのスペックと結果を並べてみる。

項目 CoreBook X (偽装) 自作PC (真実) 倍率
CPU Ryzen 5 7430U (実体 5500U) Ryzen 7 7800X3D 約3倍
GPU Radeon Graphics (内蔵) RTX 4070 Ti Super 16GB 約10倍以上
VRAM 512MB (共有) 16GB (専用) 32倍
Monitor 14インチ 60Hz 43インチ 144Hz (ミニLED) 測定不能

数字だけを見れば、ただの「高いPCを買っただけ」に見えるかもしれない。 しかし、私にとって重要なのは、その「差」の大きさではない。 「その数字が、100%の信頼に基づいているか」という一点だ。

CoreBook Xの400 ptsは、嘘の上に築かれた、崩れかけの400 ptsだった。 今日の1,120 ptsは、私がネジ一本から締め直し、環境を整え、パーツたちと対話して導き出した、揺るぎない1,120 ptsだ。 この数字の重みは、天と地ほどの差がある。

自作PCは、時に面倒で、時に高くつく。 しかし、その対価として得られるのは、誰にも、何ものにも汚されない「真実の掌握」だ。 私は、もう二度と「メーカーの言いなり」にはならない。 自分の道具の性能は、自分で決め、自分で証明する。 それが、偽装事件を経て私が辿り着いた、新しい「ガジェット道」の極意だ。

5. 結び：さらば、偽装の季節。ようこそ、真実の時代。

4月1日から始まった、この10日間の記録。 偽りのハードウェアとの決別から始まり、パーツの選定、組み立て、そしてこのベンチマーク。 それは、私という一人のユーザーが、再び技術を信じるための「再生の物語」でもあった。

私のデスクには今、Hisense 43 U7Nの広大な光が溢れている。 ケースの中では、誠実なパーツたちが静かに、しかし力強く息づいている。 3月末の、あの胸を締め付けるような怒りと絶望は、もう、この圧倒的なパフォーマンスの前に霧散した。

叫ぼう。 「これが、本物だ！」
