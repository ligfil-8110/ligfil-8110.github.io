CoreBook Xをパーソナルクラウドの心臓に：Plexメディアサーバー構築とRyzen 5のトランスコード性能

はじめに：ノートPCを「サーバー」にするという贅沢

皆さん、こんにちは。CoreBook X（Ryzen 5 7430Uモデル）を手にしてからというもの、その驚異的なコストパフォーマンスと、3:2ディスプレイがもたらす圧倒的な生産性に惚れ込み、日々のメインマシンとして酷使しています。しかし、このマシンのポテンシャルは、単なる「持ち運び用のPC」に留めておくにはあまりにも勿体ない。

今日から始まる新たな連載シリーズでは、CoreBook Xをさらに深く、マニアックに使い倒すためのテクニックを紹介していきます。その第一弾として選んだテーマは、「CoreBook Xをパーソナルクラウド（Plex Media Server）の心臓にする」という計画です。

「ノートPCでサーバー？」と思われるかもしれません。しかし、近年のモバイルCPU、特にRyzen 5 7430Uのような電力効率と多コア性能を両立したプロセッサを搭載したマシンは、静音サーバーとして理想的な特性を持っているのです。今回は、CoreBook Xをメディアサーバー化するメリットと、その核心となる動画変換（トランスコード）性能について、3,200文字を超える熱量で語り尽くします。

Plex Media Serverとは何か：究極の自前ストリーミング環境

まず、Plex Media Server（以下Plex）について簡単におさらいしておきましょう。Plexは、自分のPCやNASに保存されている動画、音楽、写真などのメディアファイルを管理し、スマートフォン、タブレット、スマートTV、さらには外出先のデバイスへストリーミング配信するためのプラットフォームです。

いわば、「自分専用のNetflix」を構築できるツールです。 - 整理能力: 複雑なファイル名でも、映画のポスター、キャスト情報、あらすじなどを自動で取得し、美しくライブラリ化してくれます。 - マルチデバイス対応: 家のテレビで観ていた映画の続きを、電車の中でスマホから観る、といったシームレスな体験が可能です。 - トランスコード: 配信先の通信環境や再生能力に合わせて、動画の解像度や形式をリアルタイムで変換してくれます。

この「トランスコード」こそが、サーバー側のCPUパワーを最も必要とする部分であり、CoreBook XのRyzen 5 7430Uが真価を発揮する舞台なのです。

Ryzen 5 7430Uのトランスコード性能：6コア12スレッドの力

CoreBook Xに搭載されているRyzen 5 7430Uは、Zen 3アーキテクチャを採用した6コア12スレッドのプロセッサです。最大ブーストクロックは4.3GHzに達し、マルチスレッド性能において同価格帯のノートPCを圧倒しています。

ソフトウェア・トランスコードの検証

Plexで動画を配信する際、再生側のデバイスがその動画形式（例えば4K H.265など）を直接再生できない場合、サーバーはリアルタイムで動画を変換する必要があります。

実際に、CoreBook X上でPlexサーバーを走らせ、以下の条件でテストを行いました。 - ソース: 4K HEVC (H.265) 10bit, ビットレート約60Mbps - 変換先: 1080p 8Mbps (H.264) - 環境: Windows 11, Plex Media Server v1.40.x

Ryzen 5 7430Uは、ソフトウェア・トランスコード（CPUのみでの変換）において、1ストリームの4K→1080p変換を余裕でこなしました。CPU使用率は一時的に50〜60%程度まで上昇しますが、バッファリングが追いつかなくなるようなことは一度もありませんでした。これは、12スレッドという多コア構成が、並列処理を効率的に捌いている証拠です。

1080pから720pへの変換であれば、同時に3〜4デバイスへの配信も現実的です。家族が別々の部屋で、それぞれ異なる動画を視聴するようなシチュエーションでも、CoreBook X一台で完璧にコントロールできるのです。

ハードウェア加速（VCN）の活用

さらに、Plex Pass（有料プラン）を契約していれば、Ryzenの内蔵グラフィックスに搭載されているハードウェアエンコーダー「Video Core Next (VCN)」を利用したハードウェア加速が可能です。これを利用すると、CPUへの負荷を劇的に抑えつつ、より高効率な変換が可能になります。Ryzen 5 7430UのRadeon Graphicsは、H.264やHEVCのエンコード/デコードにおいて非常に洗練されており、画質の劣化も最小限に抑えられています。

大容量外部ストレージとの連携：CoreBook XのI/Oを活かす

メディアサーバーを構築する上で、避けて通れないのが「ストレージ容量」の問題です。CoreBook Xの内蔵SSD（512GBや1TB）だけでは、高画質な映画コレクションを収めるには心もとないでしょう。

ここで活躍するのが、CoreBook Xのフル機能USB-Cポートです。

USB-Cハブと外付けHDD/SSDの構成

私はCoreBook Xに、セルフパワー式のUSB-Cドッキングステーションを接続し、そこから複数の大容量外付けHDDを繋いでいます。 - 接続安定性: サーバー運用において最も重要なのは、ストレージとの接続が切れないことです。安価なバスパワーハブではなく、ACアダプタから給電するドッキングステーションを使用することで、HDDの電力不足による切断を防いでいます。 - データ転送速度: USB 3.2 Gen 2の帯域があれば、複数の動画を同時に読み書きしてもボトルネックになることはありません。4K動画のストリーミング中であっても、バックグラウンドでのファイル整理がスムーズに行えます。

ネットワーク共有（SMB）のコツ

Plexだけでなく、PCからファイルを直接操作できるよう、Windowsの共有機能（SMB）も併用しています。CoreBook Xをメディアサーバー兼、簡易的なNASとして活用するわけです。Windows 11の設定で、共有フォルダのアクセス権限を適切に設定し、ネットワーク上の他のデバイスから固定IP（またはホスト名）でアクセスできるようにしています。

静音性と省電力：CoreBook Xがサーバーに向いている理由

「サーバー」と聞くと、轟音を立てて回るファンや、莫大な電気代を想像するかもしれません。しかし、CoreBook Xには従来のサーバー機にはない、ノートPCならではの利点があります。

圧倒的な静音性

CoreBook Xの冷却システムは非常に優秀です。Ryzen 5 7430UはTDP 15W（設定により調整可能）と低消費電力であるため、通常のアイドル時や、低ビットレートの配信中であれば、ファンはほとんど回転しません。夜静かなリビングに置いておいても、その存在を忘れてしまうほどです。

常時稼働とスリープ運用のバランス

ノートPCであるCoreBook Xは、停電時のバックアップバッテリー（内蔵バッテリー）を標準装備しているようなものです。急な停電でも、サーバーが即座にダウンしてファイルシステムが破損するリスクが低いのは、大きなアドバンテージです。

私は通常、CoreBook Xを「カバーを閉じた状態で何もしない」設定にし、ディスプレイ出力をオフにした状態で運用しています。また、Plexの「Wake on LAN」機能を活用すれば、アクセスがない時はスリープさせ、視聴を開始しようとした時に自動で起動させることも可能です。これにより、電気代を抑えつつ、必要な時だけ即座にサービスを提供できる、賢いサーバー運用が実現します。

設定のポイント：Windows 11での最適化

CoreBook XでPlexを安定稼働させるために、いくつか施している設定を紹介します。

  1. 電源プランの調整: 「高パフォーマンス」を選択しつつ、詳細設定で「ハードディスクの電源を切る」を「なし」に設定します。これにより、ストリーミング開始時のラグを防ぎます。
  2. 自動ログインの設定: Windowsが再起動した際（アップデート後など）、パスワード入力なしでデスクトップが表示され、Plexサービスが自動起動するように設定しています（netplwizを使用）。
  3. ディスプレイ出力オフ: 外部モニターを繋いでいない状態でも、リモートデスクトップで操作しやすいよう、仮想ディスプレイアダプタなどのツールを導入すると、ヘッドレス（モニターなし）運用が捗ります。

おわりに：CoreBook Xが暮らしのハブになる

CoreBook XをPlexメディアサーバーの心臓にする。それは、単に動画を観るための仕組みを作る以上の意味を持っています。それは、この小さな、しかしパワフルなノートPCを、自分のデジタルライフ全体の中心点（ハブ）として位置づける行為です。

Ryzen 5 7430Uの確かな性能、3:2ディスプレイがもたらす設定作業のしやすさ、そしてノートPCとしての洗練された設計。これらが組み合わさることで、専門的なサーバー機器を買うよりも、はるかに身近で、かつ高性能なメディア環境を手に入れることができました。

家中どこにいても、お気に入りの映画や音楽が、CoreBook Xから届けられる。その快適さを一度知ってしまうと、もう元には戻れません。もし、あなたの手元に持て余しているCoreBook Xがあるなら（あるいはメインマシンのパワーに余裕があるなら）、ぜひPlexの導入を検討してみてください。

明日の記事では、このメディアサーバーの管理や、さらなるマルチタスクを加速させる「4Kモニター接続とデュアルディスプレイ術」についてお話しします。CoreBook Xの視界を物理的に広げることで、何が変わるのか。楽しみにしていてください。
